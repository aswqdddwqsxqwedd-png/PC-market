# Адаптивные стили для мобильных устройств

## Описание изменений

Были добавлены адаптивные стили (CSS media queries) ко всем HTML-файлам фронтенда для обеспечения корректного отображения на мобильных устройствах и устройствах с маленьким экраном.

## Файлы, в которые были внесены изменения

1. `/workspace/static/index.html` - основной сайт
2. `/workspace/static/admin.html` - админ-панель
3. `/workspace/static/seller.html` - панель продавца

## Технические детали

Поскольку React-компоненты используют inline-стили, были созданы CSS-селекторы, которые переопределяют эти стили при помощи:

- Атрибутных селекторов (`[style*="..."]`)
- Псевдокласса `!important` для повышения специфичности
- Media queries для разных размеров экрана

## Брейкпоинты

- **1400px** - начало адаптации для более узких экранов
- **1200px** - адаптация сеток и карточек
- **768px** - планшетная адаптация (основные изменения)
- **480px** - мобильная адаптация (дополнительные улучшения)

## Что было адаптировано

- **Шапка сайта**: переход от горизонтального к вертикальному расположению элементов
- **Навигация**: перенос кнопок в несколько строк, уменьшение отступов
- **Сетки карточек**: изменение числа колонок в зависимости от ширины экрана
- **Заголовки**: уменьшение размеров для лучшей читаемости на мобильных
- **Формы**: адаптация под мобильные экраны
- **Таблицы**: обеспечение прокрутки при необходимости
- **Модальные окна**: изменение размеров и отступов
- **Чат интерфейсы**: адаптация высоты и ориентации

## Результат

Теперь все страницы проекта корректно отображаются на мобильных устройствах, с сохранением функциональности и удобства использования.
